Objective: Create a logic circuit for an Alarm system which turns on when teh car is in motion if:
              The driver is present and seatbelt is unfastened
                                  OR
              The passenger is present and seatbelt is unfastened
                                  AND
                            The ignition is ON

Data types: 
            Inputs:
                    DRIV = Signifies the presence of the driver
                    PASS = Signifies the presence of the passenger
                    BELTD = Signifies the status of the driver's seatbelt
                    BELTP = Signifies the status of the passenger's seatbelt
                    IGN = Signifieas the status of the car's ignition
            OUTPUT:
                    Alarm = Shows whether the alarm will turn on or not.

Problem solving process: 
                        First we identified the propbelem, and sorted the data for a better understanding of what information we were                                provided with.
                        Secondly, a design layout was created on basis of which we created the truth table for the problem.
                        The truth table gave us the conditions we needed for the alarm turn on, and using these boolean expressions the                             flowchart was created, for a better undersdtanding on the full process from the beginning to the end.
                        The logic circuit was created using the boolean expressions we found using the truth table.
